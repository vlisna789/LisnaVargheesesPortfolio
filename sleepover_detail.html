Sleepover

The Sleepover app is a design made for a dare based game. My friend and I organized a pilot idea and pilot game to put together whether this type of app could be a powerful tool in already fun game.

Game Concept

The inspiration for this app came from a (horrible) movie I saw. The premise of the movie revolved around the completion of a game. In the movie, two teams have to complete a list of dares before midnight. This app is based on that idea with a couple more features.


Features

Based on the rules of the game, the features of the app are meant to extend this consciousness. Within the app, the user has the opportunity to join a team based on an access code. Afterwards they create the team, check off their dares and send pictures all the while being timed and checking the progress of other teams.


About the Interface

Happy Path


This application has 5 tabs: checklist, notifications, home, team profile and settings. I created these specific tabs because I figured the user would want easy access to settings, information on the team profile page and the checklist. I gave notifications a tab because most apps have a notifications tabs separate, so I included in that way to make the user familiar. I lastly included the home tab so users could compare themselves against the competition.
